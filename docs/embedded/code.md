The code for my iot device is made in platform io. 

I decided to go with PlatformIO over Arduino and Espressif for a few key reasons that resonated with my needs and preferences. Firstly, PlatformIO offers a more streamlined and versatile development environment. Unlike Arduino, which can sometimes feel a bit restrictive in terms of supported hardware and libraries, PlatformIO provides compatibility with a wide range of development boards and frameworks, allowing me to work with different microcontrollers beyond just Espressif's offerings.

Moreover, PlatformIO's integrated development environment (IDE) is robust and feature-rich, offering advanced debugging tools, built-in unit testing support, and seamless integration with popular version control systems like Git. This comprehensive toolset not only enhances my productivity but also ensures code reliability and maintainability, which are crucial aspects of any embedded project.
